

### Quelques opréations utiles sur les chaînes de caractères

Combinaison de chaînes de caractères

~~~ python
>>> a = 'Hello'
>>> b = 'World'
>>> a+b
'HelloWorld'
>>> a+' '+b
'Hello World'
>>> print("%s %s" % (a,b))
Hello World
~~~

Strip, upper et lower

~~~ python
>>> x
'  heLLo '
>>> print(x)
  heLLo 
>>> print(x.upper())
  HELLO 
>>> print(x.capitalize())
  hello 
>>> print(x.lower())
  hello 
>>> print(x.strip())
heLLo
>>> # Et en plus ça se chaine...
>>> print(x.strip().capitalize())
Hello
~~~
